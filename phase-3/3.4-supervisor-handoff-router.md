### 3.4 — Supervisor Handoff Router

> Concept: A supervisor agent routes work to specialized agents and hands state across them without losing task context.
> Attack surface: Handoff loops, wrong specialist selection, and cross-specialist context leakage can degrade or expose sensitive flows.
> Eval target: Specialist route accuracy > 0.88 on mixed-domain requests

---

## Section 1 — Core Intuition

One general-purpose prompt pretending to be a billing expert, a security reviewer, and a product support rep at once usually produces a blend of mediocrity. The mechanism of a supervisor router is to classify the task, dispatch to a specialist, and preserve just enough shared state for the handoff to remain coherent; the analogy is an incident commander assigning work to the right responder instead of shouting all instructions at everyone on the bridge.

ATTACK: Handoff loop
VECTOR: The supervisor keeps reassigning the same request across specialists without a clear terminal rule.
EXPLOIT PAYLOAD: mixed-domain prompt bouncing between billing and security.
CONSEQUENCE: Latency spikes and no final answer.
MITIGATION:

The route set is a fixed Python `Literal` type. Even if the model returns a novel route name, the Pydantic validator rejects it before dispatch:
```python
class HandoffState(BaseModel):
    route: Literal["billing", "security", "docs", "none"] = "none"
# Routes are dispatched through a fixed handler dict:
HANDLERS = {"billing": billing_agent, "security": security_agent, "docs": docs_agent}
handler = HANDLERS.get(state.route)  # KeyError impossible — route is validated
```

## Section 6 — Pitfalls & Pro-Tips

• FAILURE: Teams added specialists but left the routing prompt underspecified, so the supervisor guessed inconsistently.
FIX: `route: Literal["billing", "security", "docs", "none"]`
• FAILURE: Unsupported requests fell through to a generic specialist and produced confident nonsense.
FIX: `state["route"] = "none"`
• FAILURE: Specialist nodes rewrote shared state in incompatible ways and broke subsequent resumes.
FIX: `class HandoffState(TypedDict): ...`
• FAILURE: Mixed-domain prompts were evaluated only on final prose, masking route-selection regressions.
FIX: `assert mean(scores) > 0.88`
• FAILURE: Process restarts forgot in-flight handoff state, and requests restarted from the wrong specialist.
FIX: `compiled_graph = graph.compile(checkpointer=SqliteSaver.from_conn_string("supervisor_router.sqlite3"))`
